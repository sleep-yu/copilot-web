useState Hook

useState 是 React 中用于在函数组件添加状态的 Hook。

基本语法：

const [count, setCount] = useState(0);

特点：

  • 返回一个状态值和一个 setter 函数
  • 状态更新会触发组件重新渲染
  • 可以保存任何数据类型
提示：使用 useState 时，避免频繁更新对象状态，可能影响性能。
