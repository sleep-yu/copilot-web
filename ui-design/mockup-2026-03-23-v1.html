帮我解释一下什么是 React Hooks，以及 useState 和 useEffect 的基本用法

21:35

React Hooks 是 React 16.8 引入的新特性，允许你在不编写 class 的情况下使用 state 和其他 React 特性。

useState - 声明状态变量：

const [count, setCount] = useState(0);

useEffect - 处理副作用（如数据获取、订阅）：

useEffect(() => { document.title = count; }, [count]);
