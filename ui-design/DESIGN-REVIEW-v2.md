## 1. 设计方向确认

根据 PM 要求：**简约风，不完全复刻 ChatGPT**

已设计两套方案供选择：

### 方案 A: 简约现代风 ⭐ 推荐
**参考**: Claude.ai, Perplexity, Gemini 极简风格

**特点**:
- 纯净配色，大量留白
- 顶部导航栏，无侧边栏干扰
- 消息气泡轻盈，无厚重背景
- 居中内容，最大宽度 800px
- 柔和阴影，视觉通透

### 方案 B: 经典 ChatGPT 风格
**参考**: ChatGPT Web 官方界面

**特点**:
- 左右分栏布局 (侧边栏 260px)
- 消息区域深色背景区分
- 更复杂的组件层级

## 2. 简约风设计规范 (方案 A)

### 2.1 色彩系统

#### 浅色主题
| Token | Hex | 用途 |
|-------|-----|------|
| `bg-primary` | #ffffff | 主背景 |
| `bg-secondary` | #f8f9fa | 消息/输入背景 |
| `text-primary` | #1a1a1a | 主文字 |
| `text-secondary` | #6b7280 | 次要文字 |
| `text-muted` | #9ca3af | 时间戳 |
| `border-color` | #e5e7eb | 边框 |
| `accent-color` | #10b981 | 主题绿 |

#### 深色主题
| Token | Hex | 用途 |
|-------|-----|------|
| `bg-primary` | #0f0f0f | 主背景 |
| `bg-secondary` | #18181b | 消息/输入背景 |
| `text-primary` | #fafafa | 主文字 |
| `text-secondary` | #a1a1aa | 次要文字 |
| `accent-color` | #34d399 | 主题绿(亮) |

### 2.2 布局规范

```
┌──────────────────────────────────────────┐
│  [Logo] Copilot              [☀️/🌙]    │  ← 顶部导航 56px
├──────────────────────────────────────────┤
│  ┌────────────────────────────────────┐  │
│  │                                    │  │
│  │         消息区域 (居中 800px)      │  │  ← 消息最大宽度
│  │                                    │  │
│  │    [🤖] 消息气泡                   │  │
│  │                    消息气泡 [👤]   │  │
│  │                                    │  │
│  └────────────────────────────────────┘  │
│  ┌────────────────────────────────────┐  │
│  │  [ 输入框... ]              [➤]  │  │  ← 输入区域
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

### 2.3 组件规格

| 组件 | 规格 |
|------|------|
| 顶部导航 | 高度 56px, 固定定位 |
| Logo 尺寸 | 32x32px |
| 头像 | 32x32px, 圆形 |
| 消息气泡 | 圆角 16px, 内边距 12px 16px |
| 输入框 | 圆角 24px, 最大宽度 800px |
| 间距基准 | 8px 系统 |

### 2.4 动画规范

| 动画 | 时长 | 效果 |
|------|------|------|
| 消息出现 | 300ms | fadeIn + translateY(8px) |
| 按钮悬停 | 150ms | scale(1.05) |
| 主题切换 | 200ms | 颜色渐变 |
| 输入框聚焦 | 200ms | 边框变绿 + 阴影扩散 |

## 3. 深浅主题效果预览

### 浅色主题
- 纯净白底，黑字
- 薄边框分隔
- 绿色强调色点缀
- 适合白天使用

### 深色主题  
- 深灰背景 (#0f0f0f)
- 高对比度文字
- 亮绿色 accent (#34d399)
- 适合夜间使用
