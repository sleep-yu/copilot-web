## 1. 产品概述

### 1.1 产品定位
copilot-web 是为 **copilot-mini** 后端开发的 AI 聊天助手 Web 界面，采用简约现代设计风格。

### 1.2 核心功能
- AI 对话聊天
- Markdown 渲染 + 代码高亮
- 用户认证 (登录/注册)
- 会话管理 (创建/切换/删除)
- 跨设备同步 (后端存储)

## 5. API 对接

### 5.1 后端 API 基础信息

| 项目 | 值 |
|------|-----|
| 基础 URL | http://localhost:62345 |
| 认证 Header | `Authorization: Bearer <token>` |

### 5.2 API 端点

#### 认证
| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/auth/register | 注册 |
| POST | /api/auth/login | 登录 |
| GET | /api/auth/me | 当前用户 |
| POST | /api/auth/logout | 登出 |

#### 会话
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/sessions | 会话列表 |
| POST | /api/sessions | 创建会话 |
| GET | /api/sessions/:id | 会话详情 |
| PUT | /api/sessions/:id | 更新会话 |
| DELETE | /api/sessions/:id | 删除会话 |

#### 消息
| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/sessions/:id/messages | 添加消息 |
| DELETE | /api/sessions/:id/messages | 清空消息 |

### 5.3 统一响应格式

```typescript
// 成功
{ code: 0, message: "成功", data: {...} }

// 失败
{ code: 400, message: "错误描述", data: null }
```

## 6. 技术架构

### 6.1 前端技术栈
- React 19 + TypeScript
- Vite 8
- React Router (路由)
- Context API (状态管理)

### 6.2 目录结构

```
src/
├── api/              # API 请求封装
│   ├── index.ts      # 基础请求工具
│   ├── auth.ts       # 认证 API
│   └── session.ts    # 会话 API
├── components/       # 可复用组件
│   ├── Sidebar/
│   ├── Header/
│   ├── MessageBubble/
│   └── InputBox/
├── context/          # React Context
│   └── AuthContext.tsx
├── pages/            # 页面组件
│   ├── Chat.tsx
│   ├── Login.tsx
│   └── Register.tsx
├── hooks/            # 自定义 Hooks
├── utils/            # 工具函数
└── App.tsx           # 主组件
```

### 6.3 路由结构

```typescript
/                   # Chat 页面 (需登录)
/login              # 登录页面 (需游客)
/register           # 注册页面 (需游客)
```
