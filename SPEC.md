# copilot-web 产品规格文档

> AI 聊天助手 Web 界面 - 产品规格书

---

## 1. 产品概述

### 1.1 产品定位
copilot-web 是一款面向 copilot-mini 后端的 AI 聊天助手 Web 界面，为用户提供类 ChatGPT 的流畅对话体验。

### 1.2 核心价值
- 简洁优雅的对话界面
- 快速响应用户需求
- 跨设备适配能力

- 基础单位: 4px

- 侧边栏内边距: 8px
- 输入区域内边距: 16px 24px 24px

| 按钮悬停 | 150ms | ease | 背景色变化 |
| 消息出现 | 300ms | ease-out | 淡入+上移 8px |
| 加载动画 | 1.4s | ease-in-out | 打字点跳动 |
| 侧边栏滑入 | 300ms | ease | 移动端 |

## 3. 功能规格

### 3.1 核心功能

#### F1: 发送消息
- 支持 Enter 发送, Shift+Enter 换行
- 空消息不可发送
- 发送时禁用输入
- 自动滚动到最新消息

#### F2: 消息展示
- 支持 Markdown 渲染 (标题/列表/代码块/链接)
- 支持代码语法高亮
- 消息时间戳显示
- 用户/助手消息样式区分

#### F3: 主题切换
- 深色/浅色模式切换
- 记忆用户偏好到 localStorage
- 平滑过渡动画

#### F4: 新建对话
- 点击清空当前会话
- 重置输入状态

### 3.2 增强功能 (规划中)

#### F4: 会话历史
- 本地存储会话列表
- 点击切换历史会话
- 删除会话功能

#### F5: 打字机效果
- AI 回复逐字显示
- 可配置开关

#### F6: 消息操作
- 复制消息内容
- 重新生成回复

## 4. 组件清单

### 4.1 侧边栏 (Sidebar)
- 状态: 默认/收起 (移动端)
- 子组件: NewChatButton, HistoryList, ThemeToggle

### 4.2 消息行 (MessageRow)
- 状态: user / assistant
- 包含: Avatar, MessageContent, Timestamp
- 交互: 悬停显示操作按钮

### 4.3 输入框 (InputArea)
- 状态: 默认 / 禁用(加载中) / 聚焦
- 包含: Textarea, SendButton
- 自动高度调整 (1-5行)

### 4.4 加载指示器
- 三点跳动动画
- 位置: 消息区域底部

## 5. 技术架构

### 5.1 技术栈
- React 19 + TypeScript
- Vite 8
- 纯 CSS (无框架)

### 5.2 目录结构
```
src/
├── App.tsx           # 主组件
├── App.css           # 主样式
├── index.css         # 全局样式/变量
├── main.tsx          # 入口
└── utils/
    └── modelSwitch.ts # 模型切换工具
```

### 5.3 状态管理
- React useState/useRef
- localStorage 持久化 (主题)

### 5.4 API 对接
```typescript
// 请求
POST http://localhost:62345/copilot/hook
{
  "sessionId": "web-client-{timestamp}",
  "data": {
    "fromUser": "user_001",
    "type": "text",
    "content": "用户输入"
  }
}

// 响应 (预期)
{
  "data": {
    "messages": [
      { "fromUser": "system", "content": "AI回复" }
    ]
  }
}
```
